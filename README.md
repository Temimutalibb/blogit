# blogIT

blogIT is a dynamic blogging platform where users can create, publish, and share their content. The platform is designed to empower writers by allowing them to earn based on the readership of their posts.

## Features

*   **Post and Earn**: Users can publish articles and earn rewards based on engagement.
*   **Rich Content Creation**: Uses Draft.js for a rich text editing experience.
*   **Categorization**: Posts are organized into categories for easy discovery.
*   **Discover Content**: View trending and recent posts.
*   **Pagination**: Browse through a paginated list of blog posts on the home page.
*   **Social Sharing**: Share posts easily on social media.

## Tech Stack

*   **Frontend**: [React](https://reactjs.org/), [Vite](https://vitejs.dev/)
*   **Routing**: [React Router](https://reactrouter.com/)
*   **Backend & Database**: [Firebase Realtime Database](https://firebase.google.com/products/realtime-database)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Rich Text Editing**: [Draft.js](https://draftjs.org/)
*   **HTTP Client**: [Axios](https://axios-http.com/)

## Getting Started

This project uses Vite for a fast development experience.

### Prerequisites

*   Node.js and npm (or yarn/pnpm)
*   A Firebase project with Realtime Database configured.

### Installation and Running

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Create a `.env.local` file in the root and add your Firebase configuration.
4.  Start the development server: `npm run dev`
